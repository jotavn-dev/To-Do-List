# 📝 To-Do List em Python

Um gerenciador de tarefas simples, prático e executado diretamente no terminal. O projeto foi desenvolvido em Python e conta com um sistema robusto de tratamento de exceções, garantindo que o programa não feche inesperadamente caso o usuário digite algo inválido.

---

## 🚀 Funcionalidades

* **Adicionar Tarefa:** Permite cadastrar uma nova tarefa na lista.
* **Listar Tarefas:** Exibe visualmente as tarefas cadastradas e seus respectivos status de conclusão.
* **Remover Tarefa:** Exclui uma tarefa da lista com base no seu número de índice.
* **Marcar como Concluída:** Altera o status da tarefa de pendente `[ ]` para concluída `[X]`.
* **Tratamento de Erros:** O programa valida entradas de texto onde deveriam ser números e trata o encerramento forçado via teclado (`Ctrl + C`).

---

## 🛠️ Tecnologias Utilizadas

* **Python 3.x:** Lógica principal do programa.
* **Manipulação de Listas e Dicionários:** Armazenamento dinâmico das tarefas.
* **Estruturas de Repetição e Condicionais:** Controle de fluxo do menu e validações.
